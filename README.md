# ARIDO - Free Video Calls & Messaging

## المميزات الرئيسية ✨
- ☎️ مكالمات فيديو مجانية مع Agora (أول 10,000 دقيقة/شهر مجاني)
- 💬 دردشة نصية فورية مع Firebase
- 🔐 تسجيل دخول آمن وسري
- 🎨 واجهة مستخدم حديثة وسهلة الاستخدام

## التكنولوجيا المستخدمة
- **Flutter**: إطار العمل الرئيسي
- **Firebase**: قاعدة البيانات والمصادقة
- **Agora RTC Engine**: محرك المكالمات المجانية
- **Cloud Firestore**: تخزين البيانات الآمن

## التثبيت والإعداد

### المتطلبات
- Flutter SDK 3.0+
- Android Studio
- Gradle

### خطوات التثبيت
1. استنساخ المستودع
```bash
git clone https://github.com/sharq-netizen/ARIDO.git
cd ARIDO
```

2. تثبيت الاعتماديات
```bash
flutter pub get
```

3. تشغيل التطبيق
```bash
flutter run
```
